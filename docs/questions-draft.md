# SUBJECT: MIRROR — Q1〜Q9 文案（v2）

## 文体ルール

- 短文。体言止めを混ぜる
- 二人称は使わない（「あなた」と書かず、状況だけを提示する）
- ──（ダッシュ）で余韻を持たせる
- 形式：「状況描写 2〜4行 → 選択肢 3つ」
- 選択肢は動詞で終わる（「〜する。」で統一）
- 露骨な正解/不正解を作らない。どれも一理ある選択にする
- 「テスト」「検査」のような診断ワードは使わない
- 問いのラベルは付けない（状況がそのまま問いになる）

## Q1：荷物

**主な測定軸：判断（INT / OBS / THK）**

**v1 からの変更点**：「夜道の足音」→「預かった荷物」に差し替え。反応パターンだけでなく、「何を守ろうとするか」が選択肢に滲む構造にした。v2.1 で「開けないでほしい」という禁止要素を追加し、選択の重みを強化。

```
知人から荷物を預かっている。
「開けないでほしい」とだけ言われた。

翌日、届け先が変わったという連絡が来た。
──理由の説明はなかった。
```

| 選択肢 | 判断 | 距離 | 守るもの |
|--------|------|------|----------|
| a) 深く考えず、言われた通りに届ける。 | INT | — | FRM |
| b) 連絡の不自然さを確かめてから動く。 | OBS | — | SAF |
| c) 中身を確認してから、判断する。 | THK | — | ORD |

**意図**：最初の問い。不確かな依頼への反応を見る。信頼して動く（自由）、情報を集めて身を守る（安全）、自分の基準で筋を通す（秩序）。判断スタイルと「何を守るか」が自然に重なる。

## Q2：雨宿り

**主な測定軸：対人距離（APP / BND / LED）**

**v1 からの変更点**：「座り込む隣人」→「雨宿りの相手」に差し替え。ただの援助行動ではなく、「見知らぬ相手との距離感で何を守るか」が問われる設計にした。v2.1 で「急いでいる」制約を追加し、傘を渡す＝自分が濡れるという代償を明示。

```
突然の雨。軒先に駆け込むと、先客がいた。

傘はひとつ。自分にも、この先の用事がある。
```

| 選択肢 | 判断 | 距離 | 守るもの |
|--------|------|------|----------|
| a) 傘を差し出す。自分は走ればいい。 | — | APP | REL |
| b) 黙って隣に立ち、雨が止むのを待つ。 | — | BND | SAF |
| c) 一緒に使いましょうと提案する。 | — | LED | ORD |

**意図**：見知らぬ相手との距離の取り方。自分を差し出して関係を作る（REL）、距離を保ち安全圏にいる（SAF）、場を仕切って秩序を作る（ORD）。傘という具体物が、守るものの違いを可視化する。

**v1 から変更なし。**

友人の部屋で、帰りを待っている。

|--------|------|------|----------|

**意図**：自分についての情報と、友人との関係が天秤にかかる。安全を守るか、自尊心（知る権利）を守るか、関係の透明さを守るか。

## Q4：忠告

**主な測定軸：対人距離 + 守るもの**

**v1 からの変更点**：「誰の目にも間違った選択」→「うまくいかないと感じる選択」に変更。断定を避け、正しさが曖昧な状況にした。

```
親しい人が、新しい道を選ぼうとしている。

周りは止めている。──自分にも、うまくいく気がしない。
```

| 選択肢 | 判断 | 距離 | 守るもの |
|--------|------|------|----------|
| a) 正直に、不安を伝える。 | INT | LED | ORD |
| b) 黙って見守る。選ぶのはその人だ。 | THK | BND | FRM |
| c) 別の選択肢を、それとなく見せる。 | OBS | APP | REL |

**意図**：相手の選択に対する態度。正しさを伝える（秩序）、自由を尊重する（自由）、関係を壊さず誘導する（関係）。「間違い」ではなく「不安」にすることで、どの選択も対等になる。

## Q5：沈黙

**主な測定軸：判断 + 守るもの**

**v1 からの変更点**：「期限のある決断」→「会議の沈黙」に差し替え。抽象的な「決断」ではなく、具体的な場面で「守るもの」が浮かぶ設計にした。

```
会議が止まっている。誰かが言いすぎた。

沈黙が続く。全員が、次の一言を待っている。
```

| 選択肢 | 判断 | 距離 | 守るもの |
|--------|------|------|----------|
| a) 自分から口を開く。 | INT | LED | FRM |
| b) 発言した人のほうを見て、少し頷く。 | OBS | APP | REL |
| c) 話題を変えて、場を動かす。 | THK | — | ORD |

**意図**：緊張した場面での振る舞い。沈黙を破る自由（FRM）、傷ついた人を支える関係（REL）、場の秩序を回復する（ORD）。「何を守ろうとするか」が行動の方向を決める。

## Q6：手紙

**主な測定軸：守るもの（PRD / FRM / ORD）**

**v1 からの変更点**：「取り壊される場所」→「過去の自分からの手紙」に差し替え。外的な喪失ではなく、内面の葛藤で「守るもの」を浮き彫りにする。

```
引き出しの奥から、古い手紙が出てきた。

過去の自分が書いたものだ。──今の自分とは、まるで違うことが書いてある。
```

| 選択肢 | 判断 | 距離 | 守るもの |
|--------|------|------|----------|
| a) 読み返す。あの頃の自分も、自分だ。 | OBS | — | PRD |
| b) 捨てる。今の自分のほうが正しい。 | INT | — | FRM |
| c) しまい直す。いつか意味がわかるかもしれない。 | THK | — | ORD |

**意図**：過去の自分との向き合い方。誇りを持って全てを引き受ける（PRD）、過去を切り離して自由でいる（FRM）、判断を保留して秩序を保つ（ORD）。対人距離は発生しない内省的な問い。

## Q7：約束

**主な測定軸：守るもの（ORD / REL / SAF）**

**v1 から変更なし。**

```
ある人との約束を守ると、別の大切な人を傷つけることになる。

どちらにも理由がある。
```

| a) 約束を守る。先に交わした言葉は重い。 | THK | BND | ORD |
| b) 事情を打ち明けて、約束を変えてもらう。 | OBS | APP | REL |

**意図**：義理と人情の衝突。秩序（約束）を守るか、関係（正直さ）を守るか、安全（誰も傷つけない）を守るか。

## Q8：鏡

**主な測定軸：判断 + 守るもの（統合問）**

**v1 からの変更点**：状況描写を拡張。「表情が違う」だけでは守るものが見えにくかったため、鏡の中の自分が「何かを言おうとしている」ニュアンスを加え、選択肢の意味を深めた。v2.1 で「見覚えがある」を追加し、夢や抽象ではなく現実の記憶と接続させた。

```
薄暗い部屋に、大きな鏡がある。

映っているのは自分だ。──ただ、表情が違う。
見覚えがある。いつかの自分だ。
```

| 選択肢 | 判断 | 距離 | 守るもの |
|--------|------|------|----------|
| a) 近づいて、目を合わせる。 | OBS | — | PRD |
| b) 背を向ける。知らなくていいこともある。 | THK | — | SAF |
| c) 「何が見える？」と聞く。 | INT | — | FRM |

**意図**：自分自身と向き合うかどうか。タイトル「MIRROR」に直結する問い。目を合わせる（自分を受け入れる＝自尊心）、背を向ける（自分を守る＝安全）、問いかける（自分を超えようとする＝自由）。

## Q9：残響（判定対象外）

**役割：selfImage として保存。スコア集計には含めない。**

**v1 からの変更点**：「最後の問いです」「守っていたもの」という説明的な導入を廃止。Q8の余韻を引き継ぎ、問いとして自然に着地する形にした。選択肢も名詞止めに統一し、文体の流れを保つ。

```
──ここまでの問いの中で、一つだけ、
ずっと手放せなかったものがある。

それは──
```

| 選択肢 | selfImage |
|--------|-----------|
| a) 自分であること。 | self |
| b) 誰かとの距離。 | bond |
| c) 筋を通すこと。 | principle |

**意図**：本人の自己認識を記録する。Q1〜Q8 の集計結果と一致するとは限らない。その「ズレ」も含めて、結果画面に反映する余地がある。「手放せなかったもの」という表現により、守る行為を自然に暗示する。
